# PROPAI Antigravity 패키지

## 파일 구조
- templates/index.html           - PROPAI 메인 앱
- templates/propai_partner.html  - 공동중개 파트너 플랫폼
- templates/propai_shorts_studio.html - 숏츠 제작 스튜디오

## 실행 방법

### 방법 1: 파이썬 로컬 서버
```
python server.py
```
브라우저: http://localhost:8080

### 방법 2: 직접 파일 열기
templates/index.html 파일을 더블클릭
